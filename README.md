# Jaskier

Jaskier is a minimalist book logging website.

Not market ready. The only form of persistent storage is <b>localStorage</b>, which is hardly fool proof. This project was partially a proof of concept and partially a learning exercise.

## Key Points

- Pure CSS and JS: No frameworks or libraries required.

- Modern Layouts: Built with all the modern amenities of Flexbox.

- Responsive Design: Adapts seamlessly to all screen sizes, from mobile to desktop.

- Lightweight & Efficient: Minimal codebase for fast loading and easy customization.
